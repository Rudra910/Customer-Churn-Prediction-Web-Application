# Customer Churn Prediction Web Application

## Project Overview
This is a Machine Learning web application that predicts whether an e-commerce customer will churn or not.

The system uses Logistic Regression and allows users to enter customer details like age, spending, purchase frequency, rating, complaints, and returns. Based on this data, the model predicts churn risk and probability.

---

## Features
- Real-time churn prediction
- Shows churn probability
- User-friendly web interface
- Machine Learning model integration
- Flask-based web application
